# Campus-Innovate-Hackathon-2.0-GHRCOE-
🚀 SkillBook – Real-Time Collaboration Platform
📌 Overview

SkillBook is a web-based platform designed to enable users to collaborate in real-time, share knowledge, and work together efficiently.
The platform includes a collaborative notes editor, session management, and an intuitive UI for seamless teamwork.

✨ Features
📝 Real-Time Notes Collaboration
Multiple users can edit the same document simultaneously.
👥 Active Collaborators Panel
View all participants in the session.
🧠 AI Co-Pilot (UI Feature)
Suggests enhancements like diagram generation.
📂 Session Management
Track session info and active time.
🧰 Collaboration Tools
Shared Notes
Whiteboard (UI)
Video Call (UI)
💾 Export & Save Options
Export notes as PDF
Save to dashboard
🛠️ Tech Stack
Frontend
HTML5
CSS3 (Flexbox, Custom UI styling)
JavaScript (Vanilla JS)
Backend (for real-time features)
Node.js
Express.js
Socket.IO (for live collaboration)
